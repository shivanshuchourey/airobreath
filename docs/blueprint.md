# **App Name**: NoteSnap

## Core Features:

- Live Classroom Streaming: Live classroom streaming to allow parents to view their child's classroom in real-time.
- Digital Whiteboard Sync: Digital Whiteboard Sync: Classroom board synced with the app, allowing instant access for parents and an option to export notes as PDF.
- AI Study Companion: AI Study Companion tool: Auto-generates notes from classroom recordings.
- Parent-Teacher Chat: Parent-Teacher Chat: Instant and secure chat functionality for direct communication between parents and teachers, including group chats and voice message support.
- Secure Authentication: Secure user authentication to ensure that the video stream from a classroom can only be viewed by authenticated users.

## Style Guidelines:

- Primary color: HSL 48, 84, 57, converted to Hex #F0B932. A vibrant yellow to convey optimism and a sense of enlightenment, nodding to the app's educational context.
- Background color: HSL 48, 21, 94, converted to Hex #F4F0E4. This provides a light, warm base that is easy on the eyes, preventing fatigue during prolonged use.
- Accent color: HSL 18, 73, 53, converted to Hex #D45A16. This orange provides good contrast, to draw attention to calls to action and important updates or alerts.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a modern look with approachability, ensuring readability and a welcoming interface for all users.
- Icons should be simple and modern, using a thin line style to align with the clean and intuitive interface. Color accents should highlight key interactive elements.
- A clean and modern layout with a focus on ease of navigation, utilizing a grid system to maintain consistency and order.
